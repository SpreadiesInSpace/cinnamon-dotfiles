**Steps to Install**

1. Clone this repo
```bash
git clone https://github.com/SpreadiesInSpace/cinnamon-dotfiles
```
2. Head to cinnamon-dotfiles
```bash
cd cinnamon-dotfiles
```
3. bash Setup.sh to run script
```bash
# Default NixOS behavior doesn't let you chmod+x then ./Setup.sh
bash Setup.sh
```
